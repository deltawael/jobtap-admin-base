<h1>JobTap Admin Frontend</h1>

A Vue 3 + Vite admin frontend used by the JobTap Admin Base monorepo.

## Development

```bash
pnpm install
pnpm dev
```

## Type Check

```bash
pnpm typecheck
```

## Branding

Customer-facing branding has been centralized. Update these files first when preparing a customer-specific build:

- `src/config/brand.ts`
- `public/logo.png`
- `public/favicon.svg`
- `public/favicon.ico`

For the full workflow, see:

- `../docs/guide/customer-branding.md`
